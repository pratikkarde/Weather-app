
### Deployment on GitHub Pages  
1. Commit all changes and push to GitHub  
   ```sh
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```
2. Go to your GitHub repository

4. Select the branch (e.g., `main`) and click **Save**
